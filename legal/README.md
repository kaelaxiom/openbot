# Legal Documentation

This directory contains the legal documentation for OpenBot.

These documents define the terms under which the bot operates and describe how user data is handled.

## Documents Included

### 1. Terms of Service
Defines:
- Conditions of use
- User responsibilities
- Bot capabilities
- Limitation of liability
- Compliance with Discord policies

File: `TERMS.md`

### 2. Privacy Policy
Describes:
- Data collected
- How data is used
- Data storage practices
- External AI providers
- Security considerations
- User rights

File: `PRIVACY.md`

## Scope

These documents apply to any deployment of OpenBot where it interacts with Discord users.

If you self-host this project, you are responsible for:
- Reviewing these documents
- Modifying them if your deployment differs
- Ensuring compliance with applicable laws

## Discord Compliance

OpenBot is designed to operate in compliance with:

- Discord Developer Terms of Service
- Discord Community Guidelines
- Discord API Policies

Server administrators are responsible for configuring permissions appropriately.

## Important Notice

If you modify the bot to:
- Store additional data
- Integrate new external services
- Execute additional local actions

You must update the Privacy Policy accordingly.
