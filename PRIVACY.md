# Privacy Policy

Last updated: [20260217]

## 1. Overview
OpenBot respects user privacy and aims to be transparent about data handling practices.

## 2. Data Collected
The bot may collect and process:

- Discord user IDs
- Discord usernames
- Discord server IDs
- Channel IDs
- Message content
- Interaction metadata
- Logs generated during operation

## 3. Data Usage
Collected data is used to:

- Generate AI-based responses
- Maintain contextual conversations
- Improve functionality
- Monitor performance and security
- Debug operational issues

## 4. External AI Providers
Message content and related metadata may be transmitted to external AI providers (e.g., OpenAI or similar services) for response generation.

These providers process data according to their own privacy policies.

## 5. Local Execution
The bot may execute configured local commands or actions on the host system.

Server administrators are responsible for understanding and managing these capabilities.

## 6. Data Storage
Logs and conversation data may be stored persistently depending on configuration.

Data retention duration is determined by the system administrator.

## 7. Data Sharing
Data is not sold.

Data may be shared only with:
- External AI service providers required for functionality
- Hosting infrastructure providers
- Legal authorities if required by law

## 8. Data Security
Reasonable technical safeguards are implemented; however, no system can guarantee absolute security.

## 9. User Rights
Users may request deletion of stored data by contacting the bot administrator.

## 10. Discord Compliance
This bot adheres to Discord Developer Terms and Discord Privacy Requirements.

## Disclaimer
OpenBot is an independent project and is not affiliated with, endorsed by, or sponsored by Discord Inc.
