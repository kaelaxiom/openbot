# Privacy Policy

Last updated: [20260217]

## 1. Data Collected
OpenBot may receive:
- Discord user IDs
- Discord server IDs
- Message content sent in channels where the bot is active

## 2. Data Usage
Data is used solely to:
- Generate responses
- Improve bot functionality
- Provide AI-based assistance

## 3. Data Storage
OpenBot does not permanently store personal data unless explicitly configured to do so.
Logs may be temporarily stored for debugging purposes.

## 4. Data Sharing
No personal data is sold or shared with third parties, except:
- AI service providers (e.g., OpenAI) used to generate responses

## 5. Data Retention
Temporary logs are deleted periodically.

## 6. User Rights
Users may request deletion of their data by contacting the developer.

## 7. Security
Reasonable security measures are implemented to protect data.

## Contact
For privacy concerns, contact: [https://github.com/kaelaxiom]
